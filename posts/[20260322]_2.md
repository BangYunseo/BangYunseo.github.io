# [백준 1000번] A+B 문제 풀이

## 접근 방식
가장 기본적인 브론즈 5 레벨의 입출력 문제입니다.
주어진 두 정수 A와 B를 입력받아서 더한 결과를 출력하면 됩니다. 여러 언어로 간단하게 풀이할 수 있지만, 여기서는 C++를 사용했습니다.

## 해결 방법
C++의 `std::cin`과 `std::cout`을 이용해 공백을 기준으로 나뉜 두 숫자를 차례대로 입력받아 합을 구합니다.

## 소스 코드 (C++)
```cpp
#include <iostream>

using namespace std;

int main() {
    int A, B;
    cin >> A >> B;
    cout << A + B << endl;
    return 0;
}
```

## 풀이 회고
너무 기초적인 문제라 따로 막힌 부분은 없었지만, C++ 입출력 속도를 고려할 때 나중에 `ios_base::sync_with_stdio(false);` 등을 활용하는 습관을 들이는 것도 좋을 것 같습니다.
